    Tables in HTML
    This is the table data tag, it helps create a cell for the table
    colspan This attribute contains a non-negative integer value that indicates for how many columns the cell extends. Its default value is 1
    headers This attribute contains a list of space-separated strings, each corresponding to the id attribute of the 'th' elements that apply to this element.
    rowspan This attribute contains a non-negative integer value that indicates for how many rows the cell extends.
    Deprecated attributes
    abbr This attribute contains a short abbreviated description of the cell's content.
    align This enumerated attribute specifies how the cell content's horizontal alignment will be handled
    axis This attribute contains a list of space-separated strings.
    bgcolor This attribute defines the background color of each cell in a column.
    char The content in the cell element is aligned to a character.
    charoff This attribute is used to shift column data to the right of the character specified by the char attribute
    height This attribute is used to define a recommended cell height.
    scope This enumerated attribute defines the cells that the header (defined in the 'th') element relates to
    valign This attribute specifies how a text is vertically aligned inside a cell.
    width This attribute is used to define a recommended cell width.
